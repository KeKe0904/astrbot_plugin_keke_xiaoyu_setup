# Changelog

## [1.1.7] - 2026-04-05

### Added
- 增强自动监听功能，确保在QQ官方机器人和onebot平台上支持
- 支持多种消息事件监听方式，提高平台兼容性

### Changed
- 优化消息事件监听器注册逻辑，支持不同平台的不同API
- 改进日志记录，提供更详细的平台支持信息
